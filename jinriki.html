Jinriki Voicebanks

(マスクグリン ・ミンティー)


Information

Jinriki UTAUs are voicebanks for UTAU created using voice samples from a source unintended for UTAU. This can include cartoon characters, text to speech programs, celebrities, singers, etc.


DISCLAIMER

None of these voicebanks can be properly released, due to it being against the UTAU TOS to release a voicebank without the voice provider's permission. Any downloads are probably audio files without an OTO.ini (settings) file.
